HOW TO INSTALL

Use the link to install mysql-connector
https://dev.mysql.com/downloads/file/?id=480300

Install xampp and start it

Start MySQL and Apache

Press admin on MySQL

Click the SQL button

Drag and drop the carDatabase.sql file into the textbox and run it

Start up the solution in Visual Studio

Open the solution explorer, right click on references, click add reference, click extensions
and run the program.
